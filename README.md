This project is an AI-powered job verification and resume matching system. It ensures that candidates only apply for legitimate job postings and provides a success rate prediction for each candidate applying to a verified job.

Features:

Fraud Detection:

Detects fake or fraudulent job postings using Naive Bayes and SGD classifiers.

Text-based analysis of job descriptions using CountVectorizer.

Resume Analysis & Job Matching:

Parses candidate resumes using NLP techniques.

Extracts skills, education, experience.

Matches resumes with verified job postings.

Predicts a success rate for getting the job.

Technologies: Python, NLP (spaCy / NLTK), Scikit-learn, Flask, Pandas
